# CUSA Football Media Day — ESPN Production Rundown

Interactive production rundown for the 2026 Conference USA Football Media Day ESPN broadcast.

## What this is
A single-file HTML production tool containing:
- Full show rundown with inline editing
- Sponsorship dashboard (17 slots, break scheduling)
- Add/remove rows per segment
- Live total runtime calculation
- Saves progress to browser localStorage

## Live site
[View the rundown →](https://chipper-cocada-c26366.netlify.app/)

## Updating
1. Download the latest HTML from Claude
2. Open this repo, click `index.html` → ✏️ edit
3. Select all, paste new content, commit
4. Netlify redeploys in ~30 seconds

## Files
- `index.html` — main production rundown
- `assets/logos/` — CUSA and ESPN brand assets
- `assets/docs/` — source Excel and brand guide
- `archive/` — previous versions
